**One Piece Card Explorer** es una plataforma integral diseñada para coleccionistas y jugadores del TCG de One Piece. Permite la visualización, filtrado avanzado y gestión de una base de datos de cartas obtenida mediante ingeniería inversa de datos JSON.

Este proyecto ha sido desarrollado como parte de la asignatura **Desarrollo Rápido de Aplicaciones (DRA)** en la Universidad de Almería.

## 🚀 Características
- **Filtrado Dinámico:** Búsqueda por color (hexagonal), coste, tipo y set.
- **Arquitectura Robusta:** Backend escalable con Spring Boot siguiendo patrones Service-Repository.
- **Interfaz Reactiva:** Frontend desarrollado en Angular con carga perezosa (lazy loading).
- **Persistencia:** Base de datos relacional PostgreSQL para la gestión de cartas y productos.

## 🛠️ Stack Tecnológico
- **Frontend:** Angular 17+ (TypeScript, CSS3, HTML5).
- **Backend:** Java 17, Spring Boot 3 (Spring Data JPA, Hibernate).
- **Base de Datos:** PostgreSQL.
- **Herramientas:** Docker, Maven, Git.
